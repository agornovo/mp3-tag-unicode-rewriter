6. Development Environment

The software was written with OpenJDK Software Development Kit, 1.7.0_101. Netbeans 3.5.1 was used for layout generation. Ant 1.9.6 is used for the automation of source code processing. IzPack 5.0.9 is used for the preparation of the installer.

This document is prepared using Docbook XML V4.5. XMLmind XML Editor Personal Edition 7.0 was used to prepare this document. HTML and PDF conversion is processed using this editor.
