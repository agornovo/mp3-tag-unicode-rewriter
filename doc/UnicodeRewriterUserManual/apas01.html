1. Supported Encoding

Supported encoding depends on the version of the Java runtime environment. As Unicode Rewriter runs under Java Standard Edition (Jave SE) 7, all encoding supported by Java SE 7 is supported by Unicode Rewriter.

Please see the link below for the complete list of the supported encoding.
